# customer_dataset_analysis

## Project Overview
This project analyzes customer shopping behavior using Python, PostgreSQL, SQL, and Power BI.  
The dataset contains 100 customer records with information related to demographics, purchases, discounts, subscriptions, reviews, and shipping preferences.

## Tools & Technologies Used
- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Project Workflow
1. Data Cleaning and Preprocessing using Python
2. Database Integration with PostgreSQL
3. SQL Query Analysis
4. Dashboard Creation using Power BI

## Key Analysis Performed
- Revenue by Gender
- Subscriber vs Non-Subscriber Analysis
- Shipping Type Comparison
- Customer Segmentation
- Top Purchased Products
- Revenue Contribution by Age Group

## Dataset Information
- Rows: 100
- Columns: 18
